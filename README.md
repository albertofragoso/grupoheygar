# GrupoHeygar

Aplicación desarrollada para un control total de un catalogo de productos y clientes. Notificando status de cada uno de ellos. Todo en tiempo real desde cualquier dispositivo.

## Descripción

Los clientes pueden revisar en cualquier momento el proceso en el que se encuentra su producto mediante notificaciones que la aplicación envía automaticamente.

Los administradores tienen un control total de todo lo que pasa con cada producto. Sin perder ningun detalle de lo que sucede en la empresa.

## Requerimientos

- Laravel 5.0 +
- PHP 7 +
